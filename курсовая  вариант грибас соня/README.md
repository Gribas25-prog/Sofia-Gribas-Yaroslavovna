# Анализ погоды

Python‑пакет для анализа погодных данных.

## Возможности

- загрузка данных из CSV файла
- вычисление статистики
- поиск максимальных и минимальных значений
- анализ тенденций температуры
- формирование текстового отчета

## Установка

pip install .

## Запуск

analiz-pogody example_weather.csv
